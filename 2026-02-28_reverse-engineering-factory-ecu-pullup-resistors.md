# Reverse Engineering a Factory ECU to Find Sensor Pull-Up Resistor Values

*Source: rusEFI Discord, 2026-02-28 | Channel: 1356732771325968630*
*Contributors: @Robb235, @Joesphan, @jeybee, @Ognjen Galic*

## Summary

When migrating from a factory ECU to rusEFI, knowing the OEM pull-up resistor value for sensors like CLT allows the aftermarket ECU to be configured to match the factory calibration. This thread documents the methods used to determine that value by reverse engineering the factory PCB, using a combination of physical circuit tracing, multimeter measurements, and Ohm's law.

## Details

### Context

@Robb235 was reverse engineering the factory ECU from a 1999 Toyota 4Runner to determine the pull-up resistor values used for sensors such as the coolant temperature sensor (CLT). The goal was to configure the aftermarket ECU correctly so that the existing OEM sensor calibration curves remain valid.

### Method 1: Physical PCB Tracing

Tracing the CLT signal pin on the PCB revealed a 10 kΩ resistor in the signal path. The resistor lead connected to a via, which @Robb235 suspected was connected to a 5V reference pour on the other side of the board (though no traces were visible from that via on the reverse side). On late-1990s Toyota ECUs, the 5V reference is typically generated by a discrete voltage regulator — likely a TO-220 packaged linear regulator (e.g., 7805 or equivalent) rather than an integrated switching supply.

### Method 2: Voltage and Current Measurement (Ohm's Law)

@Joesphan described a measurement approach that works without knowing which node is the reference rail:

1. With the ECU powered and the sensor connected, **measure the output voltage** at the CLT signal pin relative to chassis ground.
2. **Measure the current** flowing from that pin to chassis ground using a DMM in current mode (place the meter in series by shorting the pin to ground through the meter's current input).
3. Apply Ohm's law: R = V / I to calculate the effective pull-up resistance.

This method accounts for any parallel resistances that might not be visible during PCB tracing, such as the ADC input impedance or additional filter resistors.

### Parallel Resistance Consideration

@jeybee flagged an important caution: always verify there are no other resistors in parallel with the one you have identified. Parallel paths lower the effective resistance and will give an incorrect result if only one resistor is counted. @Joesphan noted the only likely parallel path in a typical NTC circuit is the ADC input impedance to ground, which is usually very high and can be ignored in practice.

### Method 3: Resistance Measurement to Reference Rail

@Ognjen Galic suggested a static (power-off) approach: measure resistance between the CLT signal pin and the output pin of the 5V voltage regulator directly. On a board with a TO-220 regulator, the regulator's output leg is accessible and provides a fixed reference node. This gives the pull-up resistance directly without needing current measurement, provided the board has no other parallel paths.

### Pull-Up Value

@jeybee estimated the pull-up resistor in this specific Toyota ECU is likely **2.2 kΩ**, based on common OEM design conventions. The 10 kΩ component @Robb235 traced is higher than what the community typically expects for a CLT pull-up (aftermarket ECUs like Speeduino commonly use 2.49 kΩ as a de facto standard). @Joesphan noted that the 2.49 kΩ value is a Speeduino convention that propagated into the wider aftermarket community, but it is not a universal standard — the pull-up value simply needs to match the ECU's thermistor calibration table.

### Configuring rusEFI

Once the pull-up resistance is known, the matching value must be entered in the rusEFI thermistor calibration settings. The pull-up value affects the voltage-to-temperature conversion curve, so using the wrong value will produce incorrect temperature readings. rusEFI allows custom thermistor tables so any OEM pull-up value can be accommodated.

## Notes

- The specific vehicle was a 1999 Toyota 4Runner; ECU vintage affects whether discrete regulators or integrated solutions are present.
- If the CLT connector is difficult to access for in-circuit measurement, the PCB tracing + static resistance method (Method 3) is less invasive.
- Current measurement by shorting to ground only works safely on low-power reference outputs — never attempt this on a high-current rail.
