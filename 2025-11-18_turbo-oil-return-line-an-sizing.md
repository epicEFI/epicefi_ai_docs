# Turbo Oil Return Line AN Fitting Sizing

*Source: rusEFI Discord, 2025-11-18 | Channel: 1356732771325968630*
*Contributors: @Alice, @Lysdex*

## Summary

AN10 is the standard fitting size for a turbocharger oil return line. AN8 weldless bungs can be difficult to source; the recommended approach for builds that may see larger turbos in the future is to weld an AN10 bung and use an AN10-to-AN8 reducer fitting, keeping future upgrade flexibility without needing to re-weld.

## Details

### Standard AN Sizing for Turbo Oil Return

The oil return line (drain) from a turbocharger carries oil by gravity back to the sump. The fitting size at the turbo and oil pan affects flow capacity:

- **AN10** is the standard / typical size for turbo oil return lines
- **AN8** is also used and is generally sufficient for smaller turbo setups

> @Lysdex: "AN10 is standard for turbo return."

### Sourcing and Upgrade Strategy

@Alice found that **AN8 weldless oil return line bungs** were difficult to source. The alternatives considered were:

1. Weld an AN8 bung directly (requires weld-on bung, no weldless option found)
2. Weld an AN10 bung and use an AN10-to-AN8 step-down fitting

The chosen approach was option 2 — weld AN10 bungs and then use reducer fittings to AN8 for the current turbo. This provides:

- A cleaner installation using a standard, widely available AN10 bung size
- **Future upgrade flexibility:** if larger turbos are installed later, the AN10 bung can accommodate an AN10 return line directly, with no re-welding required

> "I'm going to buy AN-10 bungs, weld them on, and then if I want to go bigger I can."

### Recommendation

| Scenario | Recommendation |
|----------|---------------|
| Current small turbo, no upgrade planned | AN8 is sufficient if bung can be sourced |
| Planning for future larger turbos | Weld AN10 bung, run AN10-to-AN8 reducer now |
| AN8 weldless bung not available | Weld an AN10 bung with reducer fitting |

Using AN10 as the permanent weld point is a pragmatic choice that avoids future fabrication work if the turbo is ever upgraded.
