# Honda D16Z6 Cranking Timing and VVT Sync Error (C6675)

*Source: rusEFI Discord, 2026-02-28 | Channel: 1356732771325968630*
*Contributors: @Hydrocarbon, @ggurov*

## Summary

A user running a Honda D16Z6 (12/24 trigger wheel) on uaEFI reported excessive ignition advance during cranking and a recurring C6675 warning. ggurov identified the warning as a VVT sync position conflict and recommended flipping the crank or cam edge polarity and re-timing.

## Details

### Symptoms

- Engine cranks but does not start reliably; runs fine on the original Honda P28 ECU
- Perceived as firing with too much ignition advance during cranking
- Console warning C6675 appearing repeatedly

### C6675 Warning: VVT Sync Position Too Close to Trigger Sync

The C6675 warning is generated by this firmware condition:

```
if (absF(angleFromPrimarySyncPoint) < 7)
```

It fires when the VVT (cam) sync position is within 7 degrees of the primary trigger sync point — meaning the firmware cannot reliably distinguish between the two sync events. On a 12/24 trigger wheel this is a known risk depending on how the cam and crank signals align.

### Recommended Fix

ggurov's recommendation:
1. **Flip the edge polarity** — try the opposite edge (rising vs. falling) on either the crank input or the cam input
2. **Re-time** the trigger angle after the edge change using a timing light with plugs removed

The trigger angle offset can drift if the ECU loses sync between cranking cycles, which explains the observed behavior of the timing appearing 20° more advanced on each ignition cycle. Getting sync reliable eliminates this.

### Procedure for Solo Trigger Angle Setting

When working alone, you can crank the engine from the engine bay by hotwiring the starter directly (bypass the ignition switch). This lets one person operate the starter and observe the timing light simultaneously without needing a helper in the cabin.

## Notes

- The Honda OEM basemap included with the firmware may not have the trigger angle set correctly for all D16Z6 installations — always verify with a timing light.
- D16Z6 uses a 12-tooth primary and 24-tooth secondary (cam) wheel; the VVT sync point placement relative to the missing tooth position is critical.
- C6675 does not necessarily prevent running but is a strong indicator of unreliable sync, which will cause timing instability.
